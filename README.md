# CaseyZ - Discord CS2 Case Opener Bot

CaseyZ is a feature-rich Discord bot built in Python that brings the thrill of Counter-Strike 2 case openings directly to your Discord server. It features an integrated economy system, real-time market price scraping, and user inventory management.

## Features
* **Case Openings:** Open virtual CS2 cases and pull skins of varying rarities just like in the real game.
* **Live Market Prices:** Integrates with multiple APIs (Skinport, LootFarm, Steam Community Market, and CSFloat) to fetch accurate, real-world prices for skins.
* **Global Economy:** Users have persistent balances, can transfer money to each other, and build up a valuable virtual inventory.
* **Smart Caching:** Local JSON caching for market prices prevents API rate-limiting and ensures the bot responds instantly.

## Prerequisites
* Python 3.8 or higher
* A Discord Bot Token (Get one from the [Discord Developer Portal](https://discord.com/developers/applications))
* A CSFloat API Key

## Install
* pip install -r requirements.txt

* Run python cases.py
